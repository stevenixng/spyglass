spyglass
========

setup
-----

`$ git clone https://gitlab.com/stevenix/spyglass`

`$ cd spyglass`

`$ virtualenv venv`

`$ pip install -r requirements.txt`

`$ source venv/bin/activate`

`$ export FLASK_APP=spyglass.py`

`$ flask run`

Or if you want to run with debug mode on.

`FLASK_DEBUG=1 flask run`
